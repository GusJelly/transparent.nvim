# Transparent.nvim

A simple plugin that will make **any** colorscheme **transparent**.

## Installation

```lua
{
    'GusJelly/transparent.nvim',
}
```

## Usage

Just a normal `setup()` function call:

```lua
require("transparent").setup()
```
